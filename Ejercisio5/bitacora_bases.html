2. Biblioteca: Versión Migrada (Con Triggers)

Aquí arreglé el desmadre. Le puse llaves primarias a todo, amplié los VARCHAR, le metí la columna STOCK a LIBRO y armé un trigger para descontar el stock solito.


-- (Aquí va el código SQL de la creación de tablas mejoradas y el trigger que armé arriba)
CREATE TRIGGER trg_descontar_libro
AFTER INSERT ON PRESTAMOS
FOR EACH ROW
BEGIN
    UPDATE LIBRO SET STOCK = STOCK - 1 WHERE ID = NEW.LIBRO_ID;
END;

4. Abarrotes: Versión Migrada

Separé los productos en su propia tabla y le clavé dos triggers: uno para el inventario automático y otro para el total de la venta.


-- (Aquí va el código SQL de las tablas nuevas de abarrotes y mis dos triggers perrones)
CREATE TRIGGER trg_actualizar_total_venta
AFTER INSERT ON DETALLE_VENTA
FOR EACH ROW
BEGIN
    UPDATE VENTA SET TOTAL = TOTAL + (NEW.CANTIDAD * NEW.PRECIO_UNITARIO) WHERE ID_VENTA = NEW.ID_VENTA;
END;
